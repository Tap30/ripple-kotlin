# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial implementation of Ripple Kotlin SDK
- Multi-module architecture (core, android, spring, reactive)
- Event tracking with automatic batching and retry logic
- Pluggable adapters for HTTP, storage, and logging
- Coroutine-based async operations with thread safety
- Type-safe metadata management
- Comprehensive documentation and examples
- Maven Central publishing setup
- GitHub Actions CI/CD pipeline
